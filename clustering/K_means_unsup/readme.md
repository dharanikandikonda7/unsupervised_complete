# KMeans Customer Segmentation

This project performs customer segmentation using K-Means Clustering on IBM Customer Personality Analysis Dataset.

## Workflow

1. Data Collection
2. Data Cleaning
3. EDA
4. Feature Selection
5. Feature Scaling
6. Elbow Method
7. KMeans Training
8. Model Saving
9. Cluster Visualization
10. Streamlit Deployment

## Run

streamlit run app.py
